# LocateNCR - Smart City POI Recommendation System

A Streamlit-based frontend for the Smart City NCR project that recommends optimal locations for ATMs, Hospitals, and Malls using spatial analysis.

## Quick Start

```bash
cd streamlit_app
pip install -r requirements.txt
streamlit run app.py
```

## Requirements

- PostgreSQL 15 with PostGIS extension
- Database `smartcity_delhi` with populated tables
- Python 3.10+
